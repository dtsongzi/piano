# 低音谱表音符位置分析

## 五线谱基本结构

1. **五线谱容器**：
   - `.staff` 高度：180px
   - 左右边距：左侧40px，右侧20px

2. **标准五线谱线位置**：
   - 第一线：60px
   - 第二线：80px
   - 第三线：100px
   - 第四线：120px
   - 第五线：140px

3. **额外线位置**：
   - 上方额外线：20px, 40px
   - 下方额外线：160px, 180px

## 低音谱表音符位置计算

### 计算逻辑
- 五线谱的基本单位是"间"，每个间的高度是20px
- 音符位置从上往下依次递增20px
- 下加线和下加间位于五线谱下方

### 具体位置分析

| 音符 | 位置 | 计算逻辑 | 实际top值 |
|------|------|----------|-----------|
| G2   | 第一线 | 标准五线谱第一线 | 100px |
| F2   | 下加一间 | 第一线下方20px | 120px |
| E2   | 下加一线 | 下加一间下方20px | 140px |
| D2   | 下加二间 | 下加一线下方20px | 160px |
| C2   | 下加二线 | 下加二间下方20px | 180px |
| B1   | 下加三间 | 下加二线下方20px | 200px |

## 结论

**bass-b1的top值计算到200px是正确的**，原因如下：

1. **符合音乐notation标准**：下加三间确实位于下加二线下方一个间的距离
2. **保持一致的间距**：所有线和间之间的距离都是20px，保持了五线谱的比例
3. **超出容器是合理的**：低音谱表的最低音符需要超出五线谱容器的底部，这是正常的音乐notation实践
